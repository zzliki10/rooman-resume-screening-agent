# Resume Screening Agent – Rooman 48-Hour AI Challenge

Live Demo (local): Works perfectly on fresh environment  
GitHub: https://github.com/zzlk10/rooman-resume-screening-agent

Features:
- Upload Job Description (text/PDF)
- Upload multiple Resume PDFs
- AI ranks candidates with score (0–100)
- Shows explanation, missing skills & keyword suggestions

Tech Stack:
- Streamlit + OpenAI GPT-4o-mini + LangChain + PyPDF

To run:
```bash
pip install -r requirements.txt
streamlit run main.py
